# Sentiment Analysis of Amazon Mobile Reviews Using TextBlob

A Python-based sentiment analysis project developed for ARTI303: Programming for AI at IAU.

## Overview
This project analyzes customer sentiment from Amazon mobile product reviews using TextBlob's lexicon-based approach, classifying reviews as Positive, Neutral, or Negative.

## Features
- Automated sentiment classification using TextBlob
- Analysis of 1,451+ product reviews
- Data visualization using Matplotlib (bar charts, pie charts, polarity distribution)
- Exportable results dataset

## Technologies Used
- Python
- TextBlob
- Pandas
- Matplotlib

## Results
- Positive: 52.9%
- Neutral: 39.5%
- Negative: 7.6%
