# BookHub — Book Store Management System

A Java-based desktop application developed as a final project for CSC236: Object Oriented Programming 2 at IAU.

## Overview
BookHub is a multi-role book store management system that supports three user types: Customer, Admin, and Supplier. It provides a complete book purchasing experience with inventory management, order processing, and sales reporting.

## Features
- User authentication with role-based access (Customer, Admin, Supplier)
- Book inventory management with CSV import
- Shopping cart and order processing
- Customer review system
- Admin dashboard with sales reports
- Supplier stock management

## Technologies Used
- Java (OOP)
- Java Swing (GUI)
- JDBC
- MySQL Database
- NetBeans IDE
